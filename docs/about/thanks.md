# 维护者致谢与开源许可

## 🙏 核心依赖与致谢

DroidV4L2 站在巨人的肩膀上。我们由衷感谢以下开源项目及其社区：

*   **[GStreamer](https://gstreamer.freedesktop.org/)**: 驱动 Linux 端所有视频接收与 V4L2 推送的核心管道。
*   **[CameraX](https://developer.android.com/training/camerax)**: 助力 Android 端实现稳定、高性能的相机采集。
*   **[TinyRtspKt](https://github.com/Mice-Tailor-Infra/TinyRtspKt)**: 本项目配套的自研 RTSP 引擎。
*   **[v4l2loopback](https://github.com/umlaeute/v4l2loopback)**: 实现虚拟摄像头的关键 Linux 内核模块。
